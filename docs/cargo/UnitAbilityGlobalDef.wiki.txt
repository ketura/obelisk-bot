== Field notes ==
* '''<code>target</code>''' — which side this applies to. <code>ally</code> for unit-side buffs, <code>enemy</code> for unit-side debuffs (e.g. Olgoi's <code>enemy luck -1</code> aura). Note this column is named <code>target</code> here rather than the <code>global_target</code> it carries on the <code>UnitAbilityDef</code> model — the splinter's context disambiguates the prefix.
* '''<code>power</code>''' scales with rank/upgrade level. Base unit might have <code>power=1</code>; the upg variant pushes the same effect to <code>power=2</code>.
* '''<code>tag</code>''' identifies the global-passive family. Distinct values in 2026-05-03: <code>avatar_of_war_aura</code>, <code>black_dragon_ally_aura</code>, <code>black_dragon_enemy_aura</code>, <code>olgoi_aura</code>, <code>sunlight_cavalry_aura</code>. The tag is shared across variants of the same source unit, so it's a cheap way to find related rows.
* '''<code>affected_stat</code> / <code>affected_stat_amount</code>''' describe the stat the global passive applies. Always a single stat in the current corpus.
