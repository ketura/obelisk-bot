One unlock tier of an <code>ItemSetDef</code>. The hero unlocks the tier's bonuses when they have `<code>required_amount</code>` artifacts from the parent set equipped. Sets have 1-3 tiers each (~37 tiers across the 24 sets in the 2026-05-03 corpus).

The bot emits these inline on the parent <code>Data:ItemSet/<id></code> page. There are no individual <code>Data:ItemSetTier/…</code> pages.

== Field notes ==
* '''Primary key is <code>id</code>'''. The synthesized form <code><set_id>''tier''<ordinal></code> lets <code>Bonus.parent_id</code> join cleanly via <code>parent_type='item_set_tier'</code>.
* '''<code>set_id</code>''' is the parent — duplicated here as a column so filtering by set doesn't require parsing the composite id.
* '''<code>required_amount</code>''' values seen in the 2026-05-03 corpus: 2 (×17 tiers), 3 (×11), 4 (×7), 5 (×2), 6 (×2), 8 (×1).
* '''<code>description</code>''' carries the resolved English text inline. Translations live on a <code><nowiki>{{TranslationDef | type=item_set_tier | target_id=<id>}}</nowiki></code> row (desc-only — tiers have no name).
