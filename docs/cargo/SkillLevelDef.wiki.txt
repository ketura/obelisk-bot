One row per (skill, level) pair. ~342 rows in 2026-05-05 across 102 skills. Most production / arena / campaign skills have 3 levels (Basic / Advanced / Expert mastery); pseudo skills typically have 1.

Each level carries optional name / desc / icon overrides (rare — most levels inherit the parent skill's display fields), the list of sub-skills that become offered at that level, and the bonuses that take effect when the player reaches that level. Per-level bonuses are emitted as <code><nowiki>{{BonusDef | parent_type=skill_level | parent_id=<skill_id>_L<level> | …}}</nowiki></code> rows.

== Field notes ==
* '''Primary key is <code>(skill_id, level)</code>'''.
* '''<code>name_sid</code> / <code>desc_sid</code> / <code>icon</code>''' are sparse — populated only when the level overrides the parent skill's defaults. Most levels leave these null and inherit from the parent <code>Skill</code>. When a level ''does'' override, its resolved text for every language (English included) lives in the [[Template:TranslationDef|Translation]] table as <code>type='skill_level'</code> rows keyed by <code>target_id=<skill_id></code> + <code>variant=<level></code>.
* '''<code>offered_sub_skills</code>''' is a Cargo <code>List (,) of String</code> — the sub-skill ids the player can pick from when reaching this level. <code>HOLDS</code>-friendly: <code>WHERE offered_sub_skills HOLDS 'sub_skill_offense'</code> to find every (skill, level) that offers a given perk. Empty for skills whose levels don't offer sub-skills (some pseudo skills, some campaign-only mechanics).
* '''No <code>cost</code> column''' — unlike Law levels, skill levels don't charge a separate spend; they're earned through the hero's general level-up mechanic.
